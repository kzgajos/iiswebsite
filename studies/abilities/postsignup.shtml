Ability Measure Study Instructions

Study Instructions

Thank you for signing up for the study!

Now that you are participating in the study, there are a number of steps you will need to complete.

Steps

  • In order to participate in the study, you need to have firefox. You can install it by following this link and following the instructions on the site.
  • Open firefox and navigate to (or click the link): https://iis-secure.seas.harvard.edu/ability28.xpi. The browser will ask you to install a firefox extension. This will restart your browser, so before you click install make sure to bookmark this page or open it in a different browser.
  • Now that you have installed the extension, you will see a bar marked Abilities next to the Tools menu at the top of firefox. Open it and click Take Explicit Test. Follow the directions for taking the test.
  • Use Firefox as your primary browser and you are done.
